Khu vực đăng ký

Mình vẫn giữ phần này để sau bạn chèn form của bạn vào. Khối đã được trang trí sẵn để khi thay form thật vào thì giao diện vẫn đẹp.

  • Chèn Google Form, iframe hoặc form HTML tự làm.
  • Giữ nguyên bố cục đẹp trên nền tối.
  • Dễ sửa tiếp khi bạn có form thật.

Gợi ý dùng sau

  • Thay toàn bộ khối placeholder bên phải bằng form thật.
  • Nếu dùng Google Form thì dán iframe vào.
  • Nếu dùng form tự code thì thay nguyên nội dung khối đó.
